# podcast-generator 
#  AI Podcast Generator

 AI-powered system that converts topics into fully narrated podcast audio using automated workflows.

---

##  Features

*  AI-generated podcast scripts
*  Realistic voice generation using Murf.ai
*  Automated workflow using n8n
*  Real-time audio generation via webhooks
*  Audio output playback
*  Download audio option

---

## Workflow

User Input → Lovable UI → Webhook → n8n → AI Script Generation → Murf.ai → Audio Output

---

## Tech Stack

* Lovable (Frontend UI)
* n8n (Workflow Automation)
* AI API (Script Generation)
* Murf.ai (Text-to-Speech)
* REST APIs / Webhooks
